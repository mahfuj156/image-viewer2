# মেডিকেল ইমেজ ভিউয়ার - সম্পূর্ণ প্রজেক্ট

এটা একটা সিম্পল আর প্রফেশনাল মেডিকেল ইমেজ ভিউয়ার যেটা CT/MRI স্ক্যান দেখার জন্য বানানো হয়েছে।

## 🎯 ফিচার

- ✅ **থাম্বনেইল নেভিগেশন** - ডান পাশে থাম্বনেইল থেকে দ্রুত নেভিগেশন
- ✅ **জুম কন্ট্রোল** - জুম ইন/আউট বাটন আর মাউস হুইল সাপোর্ট
- ✅ **প্যান/ড্র্যাগ** - জুম করা ইমেজ ড্র্যাগ করতে পারবেন
- ✅ **রোটেট** - ৯০ ডিগ্রি রোটেশন
- ✅ **কীবোর্ড শর্টকাট** - দ্রুত নেভিগেশনের জন্য
- ✅ **ডার্ক থিম** - প্রফেশনাল মেডিকেল ইমেজিং ইন্টারফেস

## 📦 ইনস্টলেশন

### ধাপ ১: ডিপেন্ডেন্সি ইনস্টল করুন

```bash
npm install
```

### ধাপ ২: ডেভেলপমেন্ট সার্ভার চালু করুন

```bash
npm run dev
```

ব্রাউজারে `http://localhost:5173` খুলবে।

### ধাপ ৩: প্রোডাকশনের জন্য বিল্ড করুন

```bash
npm run build
```

প্রোডাকশন ফাইল `dist/` ফোল্ডারে জেনারেট হবে।

## 🎮 কন্ট্রোল

### মাউস কন্ট্রোল
- **লেফট ক্লিক + ড্র্যাগ**: ইমেজ প্যান করুন (যখন জুম করা থাকবে)
- **মাউস হুইল**: জুম ইন/আউট
- **থাম্বনেইলে ক্লিক**: ঐ ইমেজে জাম্প করুন

### টুলবার বাটন
- **জুম ইন**: জুম লেভেল ২৫% বাড়ান
- **জুম আউট**: জুম লেভেল ২৫% কমান
- **রোটেট**: ইমেজ ৯০ ডিগ্রি ঘুরান
- **রিসেট**: জুম, রোটেশন আর প্যান রিসেট করুন

## 🔧 কাস্টমাইজেশন

### আপনার ইমেজ যোগ করুন

`src/App.jsx` তে গিয়ে images অ্যারে মডিফাই করুন:

```jsx
const sampleImages = [
  '/path/to/your/image1.jpg',
  '/path/to/your/image2.jpg',
  '/path/to/your/image3.jpg',
]
```

### API থেকে ইমেজ লোড করুন

```jsx
import { useState, useEffect } from 'react'
import ImageViewer from './components/ImageViewer'

function App() {
  const [images, setImages] = useState([])

  useEffect(() => {
    // আপনার API থেকে ইমেজ ফেচ করুন
    fetch('/api/scans/123')
      .then(res => res.json())
      .then(data => setImages(data.images))
  }, [])

  return <ImageViewer images={images} />
}
```

### রং পরিবর্তন করুন

`src/components/ImageViewer.jsx` তে Tailwind ক্লাস মডিফাই করুন:

```jsx
// Cyan থেকে blue তে পরিবর্তনের জন্য
className="ring-cyan-500"  →  className="ring-blue-500"

// ব্যাকগ্রাউন্ড রং পরিবর্তনের জন্য
className="bg-gray-900"  →  className="bg-gray-800"
```
